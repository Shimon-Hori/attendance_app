# attendance_app
イベント時における出席管理を効率化するためのモバイルアプリ。イベント参加者が、主催者のスマートフォンにbluetoothで情報を送信するだけで出席登録ができる。このアプリにより、手作業での出席確認の手間を省くことができる。

## 言語・フレームワーク・パッケージ
- Flutter
- Dart
- sqflite
- flutter_nearby_connections
